# TikTok Wellness Blog Template

This is a simple static website template for a group blog.

## Files

- `index.html`: main blog page
- `style.css`: visual design
- `script.js`: mobile navigation menu
- `images/`: put your images here
- `assets/`: optional folder for extra files

## How to edit

1. Replace placeholder text in `index.html`.
2. Export images from Word and place them in `images/`.
3. Update image file names in `index.html`.
4. Upload the folder contents to a GitHub repository.
5. Enable GitHub Pages from repository Settings → Pages.
